# Poseidon Hash Benchmark Plan

Comprehensive benchmark comparing Poseidon1 and Poseidon2 implementations
across Solidity (EVM gas cost) and Circom (R1CS constraints + proving time).

All implementations use BN254 scalar field, x^5 S-box.
